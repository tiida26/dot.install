### 条件付きでレコードを抽出するにはその２
いろいろできる



### いろいろやってみる

レコード情報

```
mysql> select * from users ;
+----+------------+---------------------------+--------+-------+---------------------+
| id | name       | email                     | team   | score | created             |
+----+------------+---------------------------+--------+-------+---------------------+
|  1 | taguchi    | taguchi@hogehoge.com      | blue   |   5.5 | 2012-05-11 11:00:00 |
|  2 | fkoji      | fkoji@hogehoge.com        | yellow |   8.2 | 2012-06-21 12:00:00 |
|  3 | hogehoge   | hogehoge@hogehoge.com     | red    |   2.3 | 2012-06-21 13:00:00 |
|  4 | sasaki     | sasaki@hogehoge.jp        | blue   |   4.5 | 2012-06-25 14:00:00 |
|  5 | kimura     |                           | yellow |   7.4 | 2012-06-28 15:00:00 |
|  6 | tanaka     | tanaka@hogehoge.jp        | blue   |   4.2 | 2012-06-29 16:00:00 |
+----+------------+---------------------------+--------+-------+---------------------+
6 rows in set (0.00 sec)
```

`between`で範囲を指定する。

```
mysql> select * from users where score between 5.0 and 8.0;
+----+---------+------------------------+--------+-------+---------------------+
| id | name    | email                  | team   | score | created             |
+----+---------+------------------------+--------+-------+---------------------+
|  1 | taguchi | taguchi@dotinstall.com | blue   |   5.5 | 2012-05-11 11:00:00 |
|  5 | kimura  |                        | yellow |   7.4 | 2012-06-28 15:00:00 |
+----+---------+------------------------+--------+-------+---------------------+
2 rows in set (0.00 sec)
```

`in`で複数の要素を引っ掛ける

```
mysql> select * from users where team in ('yellow', 'red');
+----+------------+---------------------------+--------+-------+---------------------+
| id | name       | email                     | team   | score | created             |
+----+------------+---------------------------+--------+-------+---------------------+
|  2 | fkoji      | fkoji@dotinstall.com      | yellow |   8.2 | 2012-06-21 12:00:00 |
|  3 | dotinstall | dotinstall@dotinstall.com | red    |   2.3 | 2012-06-21 13:00:00 |
|  5 | kimura     |                           | yellow |   7.4 | 2012-06-28 15:00:00 |
+----+------------+---------------------------+--------+-------+---------------------+
3 rows in set (0.00 sec)
```

`and`で条件を追加する

```
mysql> select * from users where score >= 4.0 and team = 'blue';
+----+---------+------------------------+------+-------+---------------------+
| id | name    | email                  | team | score | created             |
+----+---------+------------------------+------+-------+---------------------+
|  1 | taguchi | taguchi@dotinstall.com | blue |   5.5 | 2012-05-11 11:00:00 |
|  4 | sasaki  | sasaki@dotinstall.jp   | blue |   4.5 | 2012-06-25 14:00:00 |
|  6 | tanaka  | tanaka@dotinstall.jp   | blue |   4.2 | 2012-06-29 16:00:00 |
+----+---------+------------------------+------+-------+---------------------+
3 rows in set (0.00 sec)
```

`or`条件を使う

```
mysql> select * from users where score >= 4.0 or team = 'blue';
```
